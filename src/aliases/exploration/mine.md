Searches the current biome for configured mining results such as ore, stone, gems, or other mineral finds.

Usage: `!mine <biome> [bonuses]`

When location-based biome lookup is enabled, `!mine [bonuses]` can infer the biome from your character's current location. When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior.

Use `!mine help` for server-aware runtime help and examples.

Configured under: `Exploration -> mine`

When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior: no matches, exactly one match, or a request to be more specific.
