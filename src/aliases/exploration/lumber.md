Searches the current biome for configured lumber and woodcutting results.

Usage: `!lumber <biome> [bonuses]`

When location-based biome lookup is enabled, `!lumber [bonuses]` can infer the biome from your character's current location. When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior.

Use `!lumber help` for server-aware runtime help and examples.

Configured under: `Exploration -> lumber`

When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior: no matches, exactly one match, or a request to be more specific.
