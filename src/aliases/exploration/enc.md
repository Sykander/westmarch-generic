Rolls a random encounter from the configured exploration encounter tables.

Usage: `!enc <biome> [bonuses]`

When location-based biome lookup is enabled, `!enc [bonuses]` can infer the biome from your character's current location. When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior: no matches, exactly one match, or a request to be more specific.

Use `!enc help` for server-aware runtime help and examples.

Configured under: `Exploration -> enc`
