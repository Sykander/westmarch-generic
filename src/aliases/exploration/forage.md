Searches the current biome for configured forage results such as herbs, food, or useful natural materials.

Usage: `!forage <biome> [bonuses]`

When location-based biome lookup is enabled, `!forage [bonuses]` can infer the biome from your character's current location. When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior.

Use `!forage help` for server-aware runtime help and examples.

Configured under: `Exploration -> forage`

When manual biome input is enabled, the first argument is matched against registered biome codes with the shared lookup behavior: no matches, exactly one match, or a request to be more specific.
