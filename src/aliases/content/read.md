Reads from a configured book or content entry and records any server-supported reading progress.

Usage: `!read <title> [-author author] [comprehend] [bonuses]`

Book titles and authors use the shared lookup behavior: no matches, exactly one match, or a request to be more specific with a short match list. Use `-author` when several configured books share a similar title or when you want to narrow by author. Add `comprehend` and roll bonuses when your server allows language comprehension checks.

Use `!read help` for server-aware runtime help.

Configured under: `Content -> read`
