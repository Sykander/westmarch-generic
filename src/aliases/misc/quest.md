Shows and manages configured quest postings for the server.

Usage:

```text
!quest
!quest completed
!quest <quest>
!quest add <quest> <note>
!quest done <quest>
!quest active <quest>
```

Quest names use the shared lookup behavior when a query is supplied. Quest encounters and quest lists must be configured by the server before players can try quest-linked content.

Use `!quest help` for server-aware runtime help.

Configured under: `Misc -> quest`
