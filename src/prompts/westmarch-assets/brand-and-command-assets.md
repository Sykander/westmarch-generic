# Westmarch visual asset prompts

Maintainer notes for generating public visual assets used by the editor, README, and Avrae embeds.

## Source and public paths

Commit source assets under:

```text
editor/public/westmarch-assets/
```

The editor build copies these into the generated top-level `public/` folder. GitHub Pages then publishes them at:

```text
https://sykander.github.io/westmarch-generic/westmarch-assets/...
```

Do not hand-edit top-level `public/`; it is generated by `npm run build` or `npx nx run editor:build`.

## Prompt file

Copy the entire prompt:

```text
src/prompts/westmarch-assets/brand-and-command-assets.prompt.md
```

Use it to generate a complete brand and embed asset set, or copy individual sections for smaller batches.

## Review checklist

- The asset remains readable at Discord embed thumbnail size.
- No copyrighted D&D marks, official monster likenesses, Discord logo, or Avrae logo.
- No watermark or fake UI text.
- Sibling command thumbnails are distinct but clearly share one system style.
- Favicon and SVG marks are simple enough to work at 16px and 32px.
- Banner and social-card crops leave safe margins for overlays and platform previews.
